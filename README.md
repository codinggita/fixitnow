# 🔧 FixItNow

**FixItNow** is a local repair service booking platform that connects users with nearby technicians for repairing electronic devices and home appliances.  
The goal of this project is to make it easier and faster to find trusted technicians without wasting time searching offline.

Users can book repair services, track repair progress, and rate technicians after the service is completed.

---

## 🚀 Features

### 👤 User Features
- Browse different repair service categories
- Book repair services for devices and appliances
- View nearby technicians with ratings and pricing
- Describe device problems before booking
- Track repair job status in real time
- Rate and review technicians after service

### 🛠 Technician Features
- Create technician profiles
- Accept or reject service requests
- Update repair job status
- Manage service availability

### 📊 Admin Features
- Manage users and technicians
- Verify technician profiles
- Manage service categories
- Monitor platform activity
